# Background Fix

- Global background sudah diganti memakai file lokal: `assets/img/background.jpg`.
- Background diterapkan ke login, landing page, dashboard, dan admin lewat override final di `assets/css/style.css`.
- File ini siap dipakai via local server, contoh:

```bash
cd icikiwir-esports-pro
python -m http.server 8000
```

Buka: `http://localhost:8000`
